Automatic Transcription of Accentuated Russian Texts
===================

dict.py
Программа читает все файлы с расширением txt, которые находятся в одной папке с программой, после чего создает словарь уникальных словоформ из всех текстов. 
Расчитана на работу с текстами с уже проставленными ударениями.

transcr.py
Программа читает все файлы с расширением txt, которые находятся в одной папке с программой, 
и создает для каждого файла новый файл с фонетической транскрипцией. Программа расчитана на работу с текстами с уже проставленными ударениями.
